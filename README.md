# Website quản lí cho cửa hàng bán pizza

### Build Web Service Using

- Front-end: 
	* HTML/CSS
	* Javascrip
	* Bootstrap
- Back-end: 
	* Java Spring boot
	* Spring data jpa
	* Spring security
	* Lombok
	* Mysql database
	* Thymeleaf
	* Spring boot devtools

### Version

* JDK 8
* Spring 2.6.6
* Esclip
